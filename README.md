## 💻 今日代码示例

# 第八题
# 编写一个程序，以逗号分隔的单词序列作为输入，按照字母顺序对每个单词进行排序，并通过逗号分隔的序列来打印单词。
# 假设向程序输入：without,hello,bag,world;则输出为：bag,hello,without,world;
# 一行代码完成所有操作
words = input("请输入以逗号分隔的单词序列: ")
print(','.join(sorted(words.split(','))))
# split() 方法：将字符串按指定分隔符分割成列表
# sort() 方法：对列表进行原地排序（修改原列表）
# sorted() 函数：返回排序后的新列表，不修改原列表
# join() 方法：将列表中的元素用指定字符串连接
